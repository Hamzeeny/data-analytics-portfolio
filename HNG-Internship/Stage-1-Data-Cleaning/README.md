# Stage 1 – Data Cleaning and Preparation

This project was completed as part of the HNG Tech Data Analytics Internship.

## Project Overview
The project focused on cleaning and preparing a raw marketing products dataset for analysis. Tasks included handling missing values, removing duplicates, standardizing data formats, and creating optimized short product titles suitable for analysis and reporting.

## Tools Used
- Microsoft Excel

## Files Included
- Cleaned dataset
- Excel workbook with formulas
- Data preparation tasks

## Key Tasks Performed
- Identified and handled missing values
- Removed duplicate records
- Standardized text and formatting
- Created SEO-friendly short titles
- Preserved formulas within the workbook

- Missing data treatment
- Duplicate removal
- Data standardization
