# Stage 2 – TradeZone SQL Business Analysis

This project was completed as part of the HNG Tech Data Analytics Internship.

## Project Overview
The project focused on cleaning, analyzing, and extracting business insights from TradeZone’s e-commerce dataset using SQL. The analysis explored customer behavior, sales performance, payment trends, seller activities, and operational insights across multiple Nigerian cities.

## Tools Used
- PostgreSQL
- pgAdmin
- SQL

## Files Included
- Data cleaning queries
- Business analysis SQL queries
- Analyst memo/report
- Cleaned database dump

## Key Tasks Performed
- Data quality assessment and cleaning
- Handling missing values and inconsistencies
- Writing analytical SQL queries
- Generating business insights for decision-making
- Creating a cleaned database export

## Business Areas Analyzed
- Sales performance
- Customer activity
- Payment trends
- Seller performance
- Regional business insights
