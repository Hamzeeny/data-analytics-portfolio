# Stage 4 – Climate Change Twitter Analysis

This project was completed as part of the HNG Tech Data Analytics Internship.

## Project Overview
The project focused on analyzing climate change discussions on Twitter using SQL and Power BI. The analysis explored public sentiment, climate stance, topic trends, aggressiveness in discussions, and patterns across different years, regions, and demographic groups using a large-scale climate Twitter dataset.

## Tools Used
- PostgreSQL
- SQL
- Power BI

## Files Included
- SQL queries and views
- Power BI dashboard PDF export
- Power BI project file (.pbix)
- Project reports and analysis documents
- Embedded public Power BI dashboard link (included in the report)

## Key Tasks Performed
- Imported and transformed large-scale Twitter data
- Cleaned and prepared datasets for analysis
- Created SQL views for efficient querying and reporting
- Analyzed sentiment and climate stance trends
- Explored topic categories and engagement patterns
- Examined aggressiveness and discourse behavior across topics
- Built interactive dashboards and visualizations in Power BI
- Generated insights from climate-related social media discussions
