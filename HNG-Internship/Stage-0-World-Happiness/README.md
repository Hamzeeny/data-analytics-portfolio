# Stage 0 – World Happiness Analysis

This project was completed as part of the HNG Tech Data Analytics Internship.

## Project Overview
The analysis explores global happiness trends using the World Happiness dataset (2015–2019). The project focused on identifying relationships between happiness scores and factors such as GDP per capita, social support, life expectancy, freedom, and perceptions of corruption.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations

## Files Included
- Excel analysis workbook
- Presentation slides (PDF)

## Key Insights
- Higher GDP per capita generally correlated with higher happiness scores.
- Social support showed a strong relationship with happiness levels.
- Regional differences in happiness trends were observed across countries.
